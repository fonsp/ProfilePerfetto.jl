ProfilePerfetto.jl

Profile your Julia code and view the results in a beautiful, interactive flame chart. 🔥

ProfilePerfetto.jl is a tiny bridge between Julia's built-in Profile standard library and the Perfetto trace viewer — the same tool used for analyzing Chrome and Android performance traces.

If you've never profiled Julia code before: don't worry. By the end of the Getting started page, you'll have your first flame chart on screen.

Why profile?

Your code is slow. You have a hunch about why. You're probably wrong. 😉

A profiler samples your program many times per second and records what function is currently running. After a few seconds of sampling, the most frequently-seen functions are almost certainly your bottleneck — no guessing required.

Installation

using Pkg
Pkg.add("ProfilePerfetto")

A 30-second example

using ProfilePerfetto

function slow_thing()
    x = 0.0
    for i in 1:10_000_000
        x += sin(i) * cos(i)
    end
    return x
end

@perfetto slow_thing()

Run that cell in Pluto, VS Code, or Jupyter, and you'll see an interactive flame chart appear right below it. Drag to pan, scroll to zoom, click on a bar to see the file and line number.

Not in a notebook?

If you're in the plain Julia REPL, use @perfetto_open instead — it opens the chart in your default browser.
